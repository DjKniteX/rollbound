📌 SPECIAL NOTES

• Spawn = players ÷ 2 (rounded up)

• Base is the primary target
• Players defend Base during Combat
• If Base = 0 → zombies target players

• Zombies scale over time
• Gang Up: 3+ zombies = +1 ATK
• Horde: 5+ zombies = +1 die

• Weapons define ACC (hit chance) and DMG (damage)
• Melee: ⚔️ + ACC vs DEF
• Ranged: ⚔️ = shots, each hits on 5+
