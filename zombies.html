📌 SPECIAL NOTES / CORE RULES

• Spawn = players ÷ 2 (rounded up)
• 2–3 players = reduced zombie pressure
• 4+ players = increased zombie scaling

• Base is the primary combat target during the Quest Phase
• During Combat Phase, all player actions defend the Base
• If Base HP reaches 0, zombies stop attacking the Base
• Zombies then target the next active player in turn order

• Zombies scale over time via HP and ATK increases
• Gang Up triggers at 3+ zombies (+1 ATK)
• Horde Pressure triggers at 5+ zombies (+1 die)
